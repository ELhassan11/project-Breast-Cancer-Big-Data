# Breast Cancer Big Data & Machine Learning Project

## Overview
This project demonstrates the use of Machine Learning and Big Data technologies for Breast Cancer classification and prediction. The notebook covers the complete workflow starting from data loading and preprocessing to model training, evaluation, and Big Data processing using PySpark.

The project is divided into multiple parts to explain both traditional Machine Learning techniques and scalable Big Data approaches.

---

# Project Features

- Breast Cancer dataset analysis
- Data preprocessing and feature scaling
- Machine Learning model training
- Performance evaluation using multiple metrics
- Confusion Matrix and ROC Curve visualization
- Big Data processing using PySpark
- Random Forest classification with Scikit-learn and Spark MLlib

---

# Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- PySpark
- Jupyter Notebook

---

# Dataset

The project uses the Breast Cancer Wisconsin Dataset available from Scikit-learn.

Dataset details:
- Binary classification problem
- Target classes:
  - Malignant
  - Benign
